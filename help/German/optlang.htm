Language Options

Language settings

This page lets you select the desired language module and localized help file from those installed. To find out how to install or even create new language packs see Language packs.

Interface
Select the desired language module to be used for menus, dialogs and messages. The English interface is built into the program, so it is always available. Other modules can be downloaded and installed separately.

Help
Select the desired help file to be used. DeskPins originally contains only the English help file.

Each option shows the language name and intended program version (e.g. German v1.30), and the corresponding file name than contains it (e.g. DeskPinsGerman.chm). The interface and help versions should match the version of the program, otherwise you may get erratic behavior.
