Language Packs

Getting language packs

You can download additional language packs from the DeskPins webpage (see Support & contact). Each language pack contains a DLL file with the translated program interface (menu, dialogs, and messages). A translated help file (CHM) will also be included, if available. Multiple languages can be installed and each user can select their favorite from the Language options page.

Installation

To install a downloaded language pack, open it with a ZIP-handling program (e.g. WinZip) and copy the included files in the DeskPins folder (which normally is "C:\Program Files\DeskPins").
