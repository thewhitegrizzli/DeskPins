Hotkeys Options

Hotkeys settings

This page configures the global system Hotkeys available with DeskPins. To change a hotkey, select a field and press the keyboard combination that you'd like to use. To clear it, hit Delete or Backspace. 

Enable
Check this to enable the hotkeys and allow them to be changed. If you need to temporarily turn off the hotkeys without clearing them, you can uncheck it.

Enter pin mode
This hotkey switches the desktop into Pin mode. The default is Ctrl-F11.

Toggle active window pin
Use this hotkey to pin or unpin the currently active window. The default is Ctrl-F12.
