Autopin Options

Autopin settings

This page configures the Autopin feature. You can learn more about it in Auto-pinning windows.

Enable
Check this to enable Autopin and allow the other options to be edited.

Rules
This list contains all the configured rules. Each rule can be specifically enabled or disabled by checking or clearing the tick box on its left. You can add a new rule by clicking the Add... button and remove any of them using the Remove button. To edit a rule, double-click on it or click the Edit... button. This will bring up the Edit Rule dialog. Finally, you can use the Move Up and Move Down buttons to reorder the rules (although their relative order doesn't affect their functionality).

Delay
This controls how long DeskPins will wait before checking the Autopin rules on a newly created window. This is necessary for rules that depend on window titles, because some windows are first created with a generic title which they change before they're shown. The default value is 200 msec. If you notice that some rules are not working, you may try increasing this value.
