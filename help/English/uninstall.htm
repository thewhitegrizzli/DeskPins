Uninstall

To uninstall DeskPins, select the Uninstall item in the DeskPins program group in the Start menu, or use the DeskPins entry in the Add/Remove Programs applet of the Control Panel.

The uninstaller will remove the program folder and the following Registry keys:

  • HKEY_LOCAL_MACHINE\Software\Elias Fotinis\DeskPins
  • HKEY_CURRENT_USER\Software\Elias Fotinis\DeskPins
