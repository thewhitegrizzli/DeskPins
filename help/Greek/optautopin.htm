Επιλογές Αυτόματου Καρφιτσώματος

Ρυθμίσεις αυτόματου καρφιτσώματος

Αυτή η σελίδα προσαρμόζει τη λειτουργία αυτόματου καρφιτσώματος. Μπορείτε να μάθετε περισσότερα γι' αυτό στο Αυτόματο καρφίτσωμα παραθύρων.

Ενεργοποίηση
Τσεκάρετε αυτό για να ενεργοποιήσετε το αυτόματο καρφίτσωμα και να μπορέσετε να ρυθμίσετε τις άλλες επιλογές.

Κανόνες
Αυτή η λίστα περιέχει όλους του διαμορφωμένους κανόνες. Κάθε κανόνας μπορεί να ενεργοποιηθεί ή να απενεργοποιηθεί ξεχωριστά, τσεκάροντας ή μη το κουτάκι στα αριστερά του. Μπορείτε να προσθέσετε έναν νέο κανόνα πατώντας το κουμπί Προσθήκη... και να αφαιρέσετε κάποιους με το κουμπί Κατάργηση. Για να επεξεργαστείτε έναν κανόνα, κάντε διπλό κλικ πάνω του ή πατήστε το κουμπί Επεξεργασία.... Τότε θα εμφανιστεί το Πλαίσιο διαλόγου επεξεργασία κανόνα. Τέλος, χρησιμοποιείστε τα κουμπιά Μετακίνηση πάνω και Μετακίνηση κάτω για να ταξινομήσετε τους κανόνες (αν και η σειρά τους δεν επηρεάζει τη λειτουργικότητά τους).

Καθυστέρηση
Αυτή καθορίζει πόσο χρόνο θα περιμένει το DeskPins πριν ελέγξει τους κανόνες αυτόματου καρφιτσώματος για ένα παράθυρο που μόλις δημιουργήθηκε. Αυτό είναι απαραίτητο για κανόνες που βασίζονται σε τίτλους παραθύρων, γιατί μερικά παράθυρα δημιουργούνται πρώτα με ένα γενικό τίτλο, τον οποίο αλλάζουν αφού εμφανιστούν. Η προεπιλεγμένη τιμή είναι 200 msec. Αν παρατηρήσετε ότι ορισμένοι κανόνες δεν λειτουργούν, δοκιμάστε  να αυξήσετε αυτή την καθυστέρηση.
